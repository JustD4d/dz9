# Анализ событий информационной безопасности

Проект для анализа и визуализации событий ИБ с использованием Python, Pandas, Matplotlib и Seaborn.

## Описание

Скрипт анализирует данные событий информационной безопасности из JSON-файла, строит графики распределения и генерирует отчеты.

## Быстрый старт

```bash
# Клонировать репозиторий
git clone https://github.com/JustD4d/python_security_analysis.git
cd python_security_analysis

# Создать виртуальное окружение
python3 -m venv venv
source venv/bin/activate  # Linux/Mac
# venv\Scripts\activate   # Windows

# Установить зависимости
pip install -r requirements.txt

# Запустить анализ
python analysis.py
